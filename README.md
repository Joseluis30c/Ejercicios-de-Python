# 📖 Descripción

Este repositorio contiene una colección de ejercicios desarrollados en
**Python** con el objetivo de practicar programación y fortalecer
habilidades en:

-   resolución de problemas
-   lógica de programación
-   fundamentos del lenguaje

Los ejercicios están pensados principalmente para **principiantes que
desean aprender Python practicando**.

------------------------------------------------------------------------

# 🎯 Objetivo del proyecto

El objetivo de este proyecto es:

-   Practicar lógica de programación
-   Mejorar habilidades en Python
-   Tener un repositorio de ejercicios resueltos
-   Servir como material de estudio

# ▶️ Cómo usar el repositorio

### 1️⃣ Clonar el repositorio

``` bash
git clone https://github.com/Joseluis30c/Ejercicios-de-Python.git
```

### 2️⃣ Entrar al proyecto

``` bash
cd Ejercicios-de-Python
```

### 3️⃣ Ejecutar un ejercicio

``` bash
python ejercicio1.py
```

# 💡 Ejemplo de ejercicio

Ejemplo simple de Python:

``` python
# Programa que suma dos números

numero1 = int(input("Ingrese el primer número: "))
numero2 = int(input("Ingrese el segundo número: "))

resultado = numero1 + numero2

print("La suma es:", resultado)
```
